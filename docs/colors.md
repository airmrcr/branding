| Color | Hex | RGB | CMYK |

| Black | `#000000` | 0, 0, 0 | 0, 0, 0, 100 |
| White | `#FFFFFF` | 255, 255, 255 | 0, 0, 0, 0 |
| Primary | `#D1FFF5` | 209, 255, 245 | 18, 0, 4, 0 |
| Secondary | `#001E18` | 0, 30, 24 | 100, 0, 20, 88 |
| Tertiary | `#016A53` | 1, 106, 83 | 99, 0, 22, 58 |
| Supplementary | `#8DB42D` | 141, 180, 45 | 22, 0, 75, 29 |
