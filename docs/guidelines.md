# Guidelines

This document contains guidelines to be followed when representing the airmrcr brand.

1. [Fonts](https://github.com/airmrcr/branding/tree/master/docs/guidelines.md#fonts)

## Fonts

The following fonts are recommended to be used when presenting text-based information, anywhere from websites to written letters:

| Type | Primary Font | Fallback Fonts |
| ---- | ------------ | -------------- |
| Monospace | [Major Mono Display](https://fonts.google.com/specimen/Major+Mono+Display?preview.script=Latn) | monospace |

For all assets, the Major Mono Display font should always be used.
